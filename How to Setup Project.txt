# рҹҡҖ Step-by-Step: Create Python Project (PyCharm-like)

## рҹ“Ғ Step 1: Create Project Folder

1. Create a folder anywhere:

   ```
   python-automation-project
   ```
2. Open it in VS Code:

   * File вҶ’ Open Folder

---

## рҹҗҚ Step 2: Create Virtual Environment (VERY IMPORTANT)

Open terminal in VS Code:

```bash
python -m venv venv
```

рҹ‘ү This creates:


---

## в–¶пёҸ Step 3: Activate Virtual Environment


## рҹ”— Step 4: Link Interpreter (like PyCharm does automatically)

1. Press **Ctrl + Shift + P**
2. Search: `Python: Select Interpreter`
3. Choose:

   ```
   venv\Scripts\python.exe
   ```

рҹ‘ү Now VS Code behaves like PyCharm project interpreter

# вҡҷпёҸ Make VS Code Feel Like PyCharm

## Enable Auto-Run Button

* Install Python extension (already done)

---

## Enable Formatting

Install:

* Black formatter (optional)

---

## Auto Activate venv (optional)

Create file:

```
.vscode/settings.json
```

Add:

```json
{
  "python.defaultInterpreterPath": "venv\\Scripts\\python.exe"
}
```

рҹ‘ү Now every time project opens вҶ’ venv auto-selected
